Registration for Vehicle Pass

Please upload the required documents in a Google Drive folder and follow these instructions:

  • Name the folder clearly and use the correct file formats (JPG or PNG only).
  • Ensure the documents are clear, readable, and scanned or captured properly.
  • Set the folder to Public View before submitting.
Documents Needed:
  • Driver’s License or Student Permit
    File name example: Drivers_License.jpg or Student_Permit.png
  • Official Receipt (OR) and Certificate of Registration (CR)
    File name example: OR.jpg and CR.png
  • Certificate of Purchase/Ownership or Deed of Sale
    File name example: CPO.jpg or DOS.png
  • ID in this institution
    File name example: ID.jpg
  • Photos of the Vehicle (all sides)
    File name example: vehicle1.jpg, vehicle2.jpg, etc.
Additional Documents (If the vehicle is not yours):
  • Owner’s Valid ID
    File name example: ValidID.jpg
  • Authorization Letter from the Owner
    File name example: Letter.jpg
